# Bambaposy
BambaPOS API Client

#### Running on local machine

1. Download and extract or clone the repo
2. Put it on a local server if you have preferences.

##### Alternatively

If you have Node.js installed: 

1. Download and extract or clone the repo
2. `$ cd` into the directory
3. `$ npm install -g live-server`
4. `$ live-server`
